Problemática

Con este proyecto se pretende crear una plataforma de los juegos más reconocidos del tipo arcade para poder ser jugados en cualquier computador.

Con este proyecto se pretende crear una plataforma de los juegos más reconocidos del tipo arcade, para que los usuarios que antes solo tenían los juegos en su respectiva consola ahora puedan volver a disfrutarlos, pero desde sus computadores.

Marco teórico

Se hizo uso de html con JavaScript puro en algunos momentos, pero una gran parte del código hace uso de P5.js con sus librerías como collide, dom., sound, etc...
